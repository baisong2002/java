## 一、JAVA的异常
>>- 概念：将程序执行中发生的不正常行为叫做异常。（开发过程中语法错误和逻辑错误不属于异常）
>>- 分类：
>>>>- 1、Error:java虚拟机无法解决的严重问题。如:JVM系统内部报错、资源耗尽等严重情况。比如：StackOverflowError和OOM。一般不编写针对性的代码进行处理
>>>>- 2、Exception:其他因编程错误或偶然的外在因素导致的一般性问题，可以使针对性代码进行处理。
>>>>- 例如：空指针访问
>>>>>>- 试图读取不存在的文件
>>>>>>- 网络连接中断
>>>>>>- 数组角标越界
>>- 处理异常：
```
    try(){
            }catch(){
            }finally{
            }
```
